# W.I.S.E.C.E.O. — Worldwide Institute of Success & Excellence

> *"Creating Equity & Opportunity"*

## About
W.I.S.E.C.E.O. is a 501(c)(3) nonprofit organization dedicated to bridging the digital divide and expanding access to technology, workforce readiness, and economic mobility for historically underserved communities worldwide.

## This Repository
This repo contains the official W.I.S.E.C.E.O. landing page — a clean, Apple-inspired single-page website with:
- Animated acronym hero section
- Grant-aligned mission copy (Apple CEI + Federal EDA)
- Programs & Services showcase
- AI-powered contact/intake form (Claude API)
- Donate & Get Involved section
- Newsletter signup

## Tech Stack
- Pure HTML/CSS/JavaScript (no frameworks)
- Claude AI API for the personalized intake form
- Hosted on Cloudflare Pages

## Deployment
This site is deployed via **Cloudflare Pages** connected to this GitHub repository.
Every push to `main` automatically republishes the live site.

## Grant Alignment
Programs are aligned with:
- 🍎 **Apple Community Education Initiative (CEI)** — Digital literacy, coding, equity in tech
- 🏛 **Federal EDA Grants** — Workforce development, entrepreneurship, economic resilience

For inquiries, visit the live site or reach out through the AI-powered contact form.
